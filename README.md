# 🔍 LLM Markdown Readiness Analyzer

A Chrome Extension that evaluates how **LLM-friendly** a webpage is — based on structure, extractability, token efficiency, and conversion readiness.

---

## 🚀 Why This Matters

Modern search is shifting from:
→ Ranking pages  
to  
→ Extracting answers

LLMs (ChatGPT, Perplexity, Gemini, AI Overviews) prefer content that is:

- Structured (Markdown-like)
- Chunked (scannable)
- Direct (answer-first)
- Efficient (low token waste)

This tool helps you **measure and fix that**.

---

## 🧠 What It Analyzes

### 🧱 Markdown Readiness
- Heading hierarchy (H1 → H2 → H3)
- Paragraph chunking
- Lists (bullets, numbered)
- Tables
- Definitions / structured patterns
- Code blocks

---

### 🧠 LLM Extractability
- Direct answer presence
- Snippet-friendly blocks
- FAQ detection
- Semantic structure
- Answer placement

---

### ⚡ Token Optimization (AI Efficiency Layer)
- Average sentence length
- Redundancy %
- Fluff density
- Token-to-information ratio

---

### 🧲 Conversion Readiness
- CTA presence
- Above-the-fold CTA
- Summary blocks
- Scannability
